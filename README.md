# Калькулятор профилей (бесплатный стек)

Готовый шаблон для публикации на GitHub Pages:
- `React + TypeScript + Vite`
- `Tailwind CSS`
- `React Hook Form + Zod`
- `Vitest`
- `GitHub Actions` для автодеплоя

## Локальный запуск

```bash
npm install
npm run dev
```

## Проверка

```bash
npm test
npm run build
```

## Публикация на GitHub Pages

1. Создайте репозиторий на GitHub и запушьте код в ветку `main`.
2. В репозитории откройте `Settings -> Pages`.
3. В `Build and deployment` выберите `Source: GitHub Actions`.
4. После пуша workflow `Deploy To GitHub Pages` соберет и опубликует сайт.

## Полезные команды

```bash
npm run dev         # разработка
npm run test        # тесты
npm run build       # production build
npm run preview     # локальный просмотр dist
```
